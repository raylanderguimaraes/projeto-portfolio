Sobre

Sou desenvolvedor Web front-end Javascript, de Serra/ES, Brasil, atualmente estudando por um curso de Desenvolvimento Web Fullstack e me especializando em algumas ferramentas como React, Typescript, jQuery, Bootstrap, para posteriormente ingressar nos conteúdos relacionados ao back-end com uma base de conhecimentos mais sólidos, onde verei as tecnologias, Firebase, NodeJs e MongoDb.

Tecnologias que Utilizo

HTML 5

Construo toda estrutura da sua página utilizando tags semânticas, dessa forma melhorando o SEO da sua página, tornando seu negócio mais fácil de ser encontrado nas buscas do Google.

CSS 3

Com o CSS podemos dar estilo a sua página deixando ele muito mais bonito, com uma aparência moderna que transmita profissionalismo, valorizando sua empresa ainda mais.

Javascript

Utilizando o JavaScript damos vida ao seu site, deixando ele muito mais dinâmico, com animações e funcionalidades de acordo com as suas necessidades.

Bootstrap

Uma ferramenta poderosa para construção de interfaces estilosas, utilizando componentes padronizados.

jQuery

Uma biblioteca JavaScript muito utilizada para criar soluções para as mais diversas aplicações Web.

React

O que há de mais moderno em tecnologia para criação de aplicações Web, Sites, e interfaces de usuário, uma ótima ferrmenta para alcançarmos ótimas performances em relação ao SEO. Além de permitir criar aplicações desde as mais simples até as mais complexas.
